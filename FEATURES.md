# 功能特色

## 核心功能

### ✅ 筆劃輸入
- 5 種基本筆劃：橫、豎、撇、點、折
- 符合標準筆順規則
- 支持 10,000+ 常用繁體字

### ✅ 連碼表
**什麼是連碼？**
取字的首筆 + 尾筆，快速輸入常用字

**例子：**
```
'的' = 撇(3) + 點(4) = '34'
'人' = 撇(3) + 點(4) = '34'
'國' = 豎(2) + 點(4) = '24'
```

**優勢：**
- 只需按 2 鍵即可輸入
- 比完整筆劃快 3-5 倍
- 覆蓋最常用的 500 字

### ✅ 智能候選
- 按使用頻率排序
- 模糊匹配
- 前綴搜索

### ✅ 詞組支持
- 常用詞組庫
- 聯想輸入
- 自動學習
